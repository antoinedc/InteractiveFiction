pawsSDB::deleteAttributes($itemName, $attributes)

Description

Deletes one or more attributes associated with the item. If all attributes of an item are deleted, the item is deleted.

If $attributes is NULL, then entire $itemName item is deleted. $attributes is an associative array, and if the value part is not NULL, then the attribute name with provided values will be deleted. If the value part is NULL, then the named attribute and all its values will be deleted. (See putAttributes for description of associative attributes array.)

Note

Because Amazon SimpleDB makes multiple copies of your data and uses an eventual consistency update model, agetAllAttributes or query request (read) immediately after a putAttributes or deleteAttributes operation (write) might not return the updated data.

Parameters

NameDescriptionRequired?
$itemName

The name of the item.

Type: String

 Yes
$attributes

An associative array of attributes.

Type: Associative array

Default: NULL, which deletes entire item $itemName.

 No

Return Value

True or false. Call getErrorCode on false.

Examples

Sample Code

    /**
     *	simple example of paws function 
     */
     ...
     include('pawsSDB.php');
     $mySDB = new pawsSDB();
     $mySDB->setDomain('NameList');
     ...
     
     $mySDB->deleteAttributes('item99');	// delete 'item99'
     
     $attr = array('Color'=>'red');
     $mySDB->deleteAttributes('item100',$attr); // delete value 'red' from attribute 'Color' of 'item100'

     $attr = array('Fish'=>NULL);
     $mySDB->deleteAttributes('item90',$attr);  // delete 'Fish' attribute of 'item90'
     ...
