Quick Start

Library comes with code samples for each available service API. To get started with the library, follow these steps:
  1. Extract the amazon-simpledb-2007-11-07-php5-library.zip file into a working directory.
  2. Edit desired sample. For example: src/Amazon/SimpleDB/Samples/CreateDomainSample.php


    • Set AWS Access Key ID and AWS Secret Access Key
      •  $AWS_ACCESS_KEY_ID        = '<Your Access Key ID>';
 $AWS_SECRET_ACCESS_KEY    = '<Your Secret Access Key>';


    • Set request parameters. For example, find following pre-generated snippet:
      •  require_once ('Amazon/SimpleDB/Model/CreateDomain.php'); 
 // @TODO: set action. Action can be passed as Amazon_SimpleDB_Model_CreateDomain 
 // object or array of parameters
 // invokeCreateDomain($service, $action);


    • And set DomainName parameter:
      •  require_once ('Amazon/SimpleDB/Model/CreateDomain.php'); 
 $action = new Amazon_SimpleDB_Model_CreateDomain();
 invokeCreateDomain($service, $action->withDomainName('MyDomain'));

  3. Run CreateDomain sample:
    4.    php src/Amazon/SimpleDB/Samples/CreateDomainSample.php
  4. You should see the output similar to the following:


    5.     CreateDomainResponse
        ResponseMetadata
            RequestId
                95cdcb68-f46c-400b-8265-8c2de2a5c475

  5. Experiment with samples. When ready, install library to your php include path and use it

Notes

You can use alternative array-based interface with the library. For example, instead of: 

    $action = new Amazon_SimpleDB_Model_CreateDomain();
    $action->setDomainName('MyDomain');
    $response = $service->createDomain($action);
You can write: 
    $response = $service->createDomain(array('DomainName' => 'MyDomain'));

Happy coding!
