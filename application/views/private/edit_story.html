Help

1. Double click on paragraphes for edit them (change the text, define if it's the start or the end of the story)
2. You can drag them, don't forget the "Save" button for saving the new position
3. Click on the red squares and drag to another paragraph for creating a link, edit or remove a link by clicking on it
4. You need to have one "start" paragraph only, that's why the first one created is set as "starting paragraph", you can change it by setting another paragraph as first with the corresponding checkbox
5. You can have as many ends as you want, even zero, but it's preferable to have one
6. The online exportation is not working now
7. For mxit, download the mxit app on Google Play or iTunes Store and add the contact "interactivefiction" for starting a story
8. Questions/Bugs/etc => adechevigne@gmail.com
