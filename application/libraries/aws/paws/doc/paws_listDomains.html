pawsSDB::listDomains($maxNumberOfDomains)

Description

The listDomains function lists all domains associated with the Access Key ID. It returns a PHP array of domain names up to the limit set by $maxNumberOfDomains.

Parameters

NameDescriptionRequired?
$maxNumberOfDomains

The maximum number of domain names you want returned.

Type: String

The range is 1 to 100.

The default setting is 100.

No

Return Value

An array of AWS domain names on success. Return NULL if no domain names defined, or bad $maxNumberOfDomains.

Special Errors

ErrorDescription
InvalidParameterValue Value for parameter $maxNumberOfDomains is invalid. $maxNumberOfDomains must be between 1 and 100.

Examples

Sample Code

    /**
     *	simple example of paws function listDomains
     */
    ...
    include('pawsSDB.php');
    $mySDB = new pawsSDB();
    ...
    $domainNameList = $mySDB->listDomains();	// get array of domain names
    if ($domainNameList == null) {
        echo("**** No domains defined ****\n");
        return false;
    } else {
	   echo("<ul>");
       foreach ($domainNameList as $domainName) {	// echo all names returned
           echo("<li>DomainName: ");
           echo($domainName . "</li>\n");
       }
       echo("</ul>\n");
    }
    ...
