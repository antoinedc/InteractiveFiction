pawsSDB utility Functions

Other than getErrorCode, these are provided for completeness if not usefulness.

pawsSDB::getErrorCode()

This returns the error code returned from AWS on the last operation.

pawsSDB::getErrorMessage()

This returns the long error message returned from AWS on the last operation.

pawsSDB::getBoxUsage()

This returns the Box Usage string returned by AWS on the last operation.

pawsSDB::getResponseID()

This returns the Response ID returned by AWS on the last operation.
