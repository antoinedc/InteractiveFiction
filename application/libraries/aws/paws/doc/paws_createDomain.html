pawsSDB::createDomain($domainName)

Description

The createDomain function creates a new domain. The domain name must be unique among he domains associated with the Access Key ID used by the pawsSDB object. The createDomain execution might take 10 or more seconds to complete.

Note

createDomain is an idempotent operation; running it multiple times using the same domain name will not result in an error response.

The new domain may not be immediately available for writing after it has been created.

You can create up to 100 domains per account.

Parameters

NameDescriptionRequired?
$domainName

The name of the domain to create.

Type: String

 Yes

Return Value

True or false. Call getErrorCode on false.

Special Errors

ErrorDescription
InvalidParameterValue Value (" + value + ") for parameter DomainName is invalid.
DomainNameNotSet The request must contain the parameter DomainName.
NumberDomainsExceeded Number of domains limit exceeded.
